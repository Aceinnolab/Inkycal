Quickstart
===========

Installing Inkycal
------------------
Install inkycal  directly via the github repo:

.. code-block:: sh
    :linenos:

    # clone the repo
    git clone -b  release/2.0.0 https://github.com/aceisace/Inky-Calendar Inkycal

    # go to downloaded repo folder
    cd Inkycal

    # install Inkycal
    pip3 install -e ./
